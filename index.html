System Setup Required

Step 1: Create Firebase Admin User

  1. Go to Firebase Console
  2. Create a new project or select existing one
  3. Go to Authentication → Sign-in method
  4. Enable Email/Password provider
  5. Go to Firestore Database and create database
  6. Click "Start collection" and create:
    • Collection name: users
    • Document ID: (auto-generated)
    • Fields: email (string), role (string = "admin")
  7. Add yourself as admin user in Authentication tab

Step 2: Test Login Credentials

Try logging in with your Firebase admin credentials.
